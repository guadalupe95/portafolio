Sobre mí

Me considero una persona proactiva, responsable y orientada a resultados, con una fuerte capacidad para trabajar tanto de manera independiente como en equipo. Destaco por mi capacidad para resolver problemas complejos de manera eficiente y con calidad del código. Mantengo aprendizaje continuo y adaptación a nuevas tecnologías.

Mi lenguaje es Java con experiencia en frameworks como Spring, Hibernate, y JUnit. Desarrollo aplicaciones web utilizando JavaEE y tecnologías front-end como HTML, CSS, y JavaScript. Manejo Bases de datos como MySQL, PostgreSQL, MongoDB y Oracle. Con control de versiones uso GitHub para la gestión de código fuente en equipos colaborativos. Tengo experiencia con metodologías ágiles como Scrum y Kanban para la gestión de proyectos. Pruebas automatizadas, unitarias y depuracion de código para garantizar la calidad del software. Dominio herramientas como Maven y Gradle.

Maria Guadalupe Mendoza Flores
